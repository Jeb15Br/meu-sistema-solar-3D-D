# Sistema Solar 3D - Setup

Este projeto pode ser executado como um site ou como um programa de computador (Desktop App).

## Opção 1: Rodar direto no Navegador (Simples)
Basta abrir o arquivo `index.html` em um navegador moderno.
*Nota: Alguns recursos como áudio ou texturas podem não carregar devido a bloqueios de segurança do navegador (CORS). Se isso acontecer, use a Opção 2 ou 3.*

## Opção 2: Servidor Local (Recomendado para Web)
Se você tem Python instalado:
1. Abra o terminal nesta pasta.
2. Rode: `python -m http.server`
3. Acesse `http://localhost:8000`

## Opção 3: Transformar em App (Electron)
Para rodar como um programa de janela:

1. Instale o **Node.js** (se não tiver): https://nodejs.org/
2. Abra o terminal nesta pasta (`Shift + Clique Direito` -> "Abrir janela do PowerShell aqui").
3. Instale as dependências:
   `npm install`
4. Inicie o App:
   `npm start`

Isso abrirá o Sistema Solar em uma janela própria!
